# NatHub v1：物理效果优化与首发计划

## 产品定位

- 核心用户：10–16 岁青少年，以及喜欢精致互动体验的成人。
- 产品表达：一座“口袋里的物理奇观馆”，优先让用户观察、触摸并发现自然现象之美，而不是完成教学任务。
- 真实性标准：基本规律必须正确，不能产生与物理相悖的反馈；光晕、色彩、声音等视觉听觉表现可以艺术化。
- 性能标准：以 iPhone 12 / A14 为最低体验基线，常态目标 60 fps；重场景可稳定降至 30 fps，但不能明显卡顿、延迟或持续过热。

## v1 范围

首发展示且可直接体验四个项目：

1. 水波
2. 光的折射
3. 沙之痕
4. 磁力阵列

首发首页不显示锁定卡片或“即将推出”。牛顿摆与镜子迷宫暂缓到后续版本，也不修改其现有功能。

## 全局体验

### UI Kit

先从现有 App 提取一份轻量 UI Kit，覆盖：

- 色彩、字体、间距与圆角
- 深色背景、玻璃和金属材质
- 普通、按下、禁用和选中状态
- 顶部导航、实验控制和设置页按钮
- 动效时长、触感强度和声音层级

后续新增界面必须复用这套视觉语言。

### 顶部按钮

- 返回与全屏改用原生 `UIBarButtonItem` 和 SF Symbols。
- 左右触控区域均为 44 × 44 pt，边距、尺寸和视觉权重对称。
- 不手绘按钮圆形或玻璃背景，由 iOS 根据系统版本生成原生外观。

### 首页与设置

- 首页保持单页实验陈列，不增加底部导航、分类、收藏或发现模块。
- 首页右上角提供设置入口。
- 设置为全 App 级，并持久保存：声音、触感、语言。
- 设置页同时提供关于、隐私政策、用户协议、技术支持、版本号与构建号。
- 实验状态不保存；每次进入都从精心设计的默认状态开始。

### 动效、触感与性能

- 不显示文字操作教程。
- 可用一次性微动效暗示交互；水波保留已有的自动引导滴水。
- 自动引导不产生触感；只有用户主动点击水面才产生轻触感。
- 拖动、划水、耙沙等连续交互不连续震动。
- 自动读取系统“减少动态效果”：保留有意义的物理运动，移除呼吸高光与装饰动效。
- 性能不足时按层级降低粒子数、光谱采样、辉光和帧率，不把核心实验变成静态画面。
- 静止画面停止刷新；只在状态改变或动画进行时渲染。

## 实验优化

### 1. 水波

核心语义是“一次点击，一滴水”。

- 用户轻点：产生一个冲击点、一组连续向外传播的波和一次声音。
- 自动引导：产生一滴水，但声音更轻且无触感；用户首次操作后永久停止本次引导。
- 修复双响：当前代码一次点击只发起一次播放，应检查并裁切音频素材本身的双峰，换成单次落水音。
- 长按：按下时只产生一次滴水；持续按住仅保持很轻的局部凹陷，不重复发声、不周期性生成波圈；松开时轻微回弹且无声音。
- 拖动：制作可开关的实验版本，形成低强度划水尾迹；与仅点击版本对比后再决定是否保留。点击滴水必须始终是视觉主角。
- 波动应表现传播、衰减和叠加；不追求完整流体模拟。

### 2. 磁力阵列

- 将“钢珠箭头”调整为可辨认的磁针阵列，磁针沿局部磁场方向旋转。
- 使用真实偶极磁场的封闭形式计算固定采样点，替换当前简化的两磁极方向近似。
- 固定约 50 个磁针，静止时停止刷新，性能成本保持很低。
- 磁针底托为单层哑光深石墨圆盘，只保留一个干净外轮廓和极弱高光，不使用双层边框。
- 磁石继续使用原有红蓝素材，并始终渲染在磁针上层，可遮挡下方磁针。
- 保持一指拖动、双指旋转，不增加旋转滑杆或教学面板。
- 视觉目标是连贯、迷人的场线秩序，而不是课堂示意图。

### 3. 沙之痕

- 采用宏观颗粒规律，不做逐粒碰撞模拟。
- 耙子推动颗粒时，前缘出现轻微堆积，后方留下连续沟槽，沟槽不会立即回弹。
- 优先使用密度场、有限粒子样本和纹理表现颗粒迁移。
- 入口图从实验内部真实粒子渲染截取，严格沿用当前入口图的构图、裁切比例与视觉重心。

### 4. 光的折射

- 保留最多三块已放置玻璃砖；默认只有一块，用户可自行添加和叠加。
- 保留三角玻璃砖、平行四边形玻璃砖和半圆玻璃砖，可拖动、旋转和移除。
- 增加光源颜色控制：白光、宽谱彩光和单色光。单色光不得凭空分解成彩虹。
- 色散优化的目标是扩大不同波长的真实出射角，使输出形成明显扇形，而不是只把一条彩色线涂亮。
- 优先通过更高色散的真实光学玻璃、棱镜角度和更长出射光程扩大扇面；辉光、饱和度与暗部对比只增强可见性，不改变光路方向。
- 平行玻璃砖在部分姿态下仍可能表现为接近一条线，这是允许且正确的结果。
- 继续使用二维逐波长光线追踪，不升级为完整三维光线追踪。

> 架构冲突：现有领域文档明确规定所有玻璃砖使用 BK7 类冕牌玻璃。改为高色散光学玻璃会改变已记录的领域决策。实现前应先确认统一材质型号，并更新 `CONTEXT.md`、ADR 与相关测试，不能悄悄偏离现有约定。

## 商业化与隐私

- v1 无账号、无广告、无第三方统计，离线优先、零数据收集。
- 摄像头画面功能不首发；移除 v1 的摄像头权限声明。
- 预埋 StoreKit 2 权益层、实验解锁状态与产品映射，但界面完全不展示价格、锁、购买、订阅或恢复购买。
- 后续使用“解锁更多实验”而非“充值”；玩法解锁优先采用非消耗型内购。

## App Store 准备

- 分类：主分类“娱乐”，副分类“教育”，不进入 Kids Category。
- 首发版本：`1.0 (1)`；版本号只在设置/关于底部展示。
- 暂定品牌：`NatHub`。
- 暂定商店名称：中文 `NatHub：物理奇境`，英文 `NatHub: Physics in Motion`。最终提交前再次检查名称占用、商标和本地化关键词。
- 提交前准备：App 图标、隐私政策 URL、支持 URL/联系方式、中文与英文商店描述、关键词、年龄分级、各目标设备截图、审核备注和隐私问卷。

## v1 完成标准

- 首页只出现四个可直接进入的实验。
- 四个实验的核心物理规则无明显错误，且用户无需阅读教程即可操作。
- iPhone 12 上没有持续掉帧、明显触控延迟或异常发热。
- 水滴一次点击只响一次；自动滴水无触感。
- 顶部按钮符合原生 iOS 外观并保持对称。
- 磁针方向与偶极磁场一致，磁石位于最上层。
- 沙粒表现出堆积、沟槽与残留。
- 白光色散形成清晰扇面；单色光不生成彩虹。
- 首发不请求摄像头权限，不展示任何付费入口或锁定内容。
